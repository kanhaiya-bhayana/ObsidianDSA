```java
public int minTime(int[] arr, int n, int k){
	int s = maxEleofArray(arr);
	int e = sumOfAllArrayElements(arr);
	
	while (s <= e){
		int mid = s + (e-s)/2;
		if (check(A,n,mid,k)){
			if (mid == 0 || check(A,n,mid-1,k)){
				return mid;
			}
			else{
				e = mid -1;
			}
		}
		else{
			s = mid + 1;
		}
	}
}

private boolean check(int [] arr, int n, int time, int k){
	int cnt = 0;
	int sum = 0;
	
	for (int i=0; i<n; i++){
		sum += arr[i];
		if (sum > time){
			cnt++;
			sum = arr[i];
			if (cnt > k){
				break;
			}
		}
		
	}
	
	return (cnt <= k);
}
```

>TC: O(n Log (e - s))
>SC: O(1)
