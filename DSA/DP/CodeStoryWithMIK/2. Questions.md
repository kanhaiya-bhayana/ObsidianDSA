
### 1. Fibonacci Series

###### Recursive
```java
f(int n)
{
	// base case
	if (n == 0 or n == 1)
	{
		return n;
	}	
	return f(n-1) + f(n-2);
}
```

> Dimensions of DP array = no. of changing variables


###### Memoization
```java
Integer[] dp;
f(int n)
{
	// base case
	if (n == 0 or n == 1)
	{
		return n;
	}	
	if (dp[n] != null)
	{
		return dp[n];
	}
	return dp[n] = f(n-1) + f(n-2);
}
```


###### Bottom Up
```java
class Solution {
    Integer[] dp;
    public int fib(int n) {
        dp = new Integer[n+1];
        return f(n);
    }

    private int f(int n)
    {
        // base case
        if (n <= 1)
        {
            return n;
        }
        dp[0] = 0;
        dp[1] = 1;
        
        for (int i = 2; i <= n; i++)
        {
            dp[i] = dp[i-1] + dp[i-2];
        }

        return dp[n];
    }
}
```

###### Constant Space
```java
class Solution {
    public int fib(int n) {
        return f(n);
    }

    private int f(int n)
    {
        // base case
        if (n <= 1)
        {
            return n;
        }
        int prev1 = 0;
        int prev2 = 1;
        
        for (int i = 2; i <= n; i++)
        {
            int curr = prev1 + prev2;
            prev1 = prev2;
            prev2 = curr;
        }

        return prev2;
    }
}
```
